אודות טוזיג

לכולנו יש אותם 'עוגנים' בחיים: תאריך ומקום הלידה, תמונה ראשונה, תמונת חתונה ועוד.
השירות שלנו מאתר את העוגנים, מלקט את הסיפורים מקליט אותם וסורק את התמונות בצורה המאפשרת לשמור ולהציג את הסיפור בפיסות נוחות, קלות לשימוש.

מה מיוחד בכלים שלנו?

פתוח

בכל עת תוכלו להוריד את כל המידע והתכנים שאספתם. בנוסף, קוד התצוגה מפותח תחת רשיון קוד פתוח וזמין ב GitHub

חכם

אנחנו עוזרים לחלק את סיפור החיים לחלקים ואז לסדר אותם כספר, סרט או כמסע סיפורי הפרוש על מסך המחשב.

חי

סיפורי החיים שאנחנו עוזרים ליצור ממשיכים להתעדכן ולהתפתח דרך תוכן מִמַּכָּרִים ומוקירים.
